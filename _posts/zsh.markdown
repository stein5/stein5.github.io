### 설치

```bash
# zsh설치
$ sudo apt-get install zsh

# og my zsh 설치
$ wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh

# zsh을 디폴트로 설정
$ chsh -s `which zsh`

# os restart
$ sudo shutdown -r 0
```
