## about tmux

### start tmux
```bash
$ tmux  # start
$ exit  # close
```

### creating named session
```bash
$ tmux new-serssion -s basic
$ tmux new -s basic
```

### command line prefix
Cntrl + b  # tell tmux that the command we're next typing is for tmux
Prefix + t  # show time
